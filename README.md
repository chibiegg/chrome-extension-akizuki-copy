# 秋月電子 買い物かごコピー

[秋月電子通商](https://akizukidenshi.com/) の買い物かごページを開いているときに、販売コードと数量をタブ区切りでクリップボードにコピーする Chrome 拡張機能です。

## 機能

- 買い物かごの全商品の**販売コード**と**数量**を取得
- **タブ区切り・1行1商品**形式でクリップボードにコピー
- ポップアップに実際のコピー内容をプレビュー表示

### コピー形式

```
107922	2
113605	3
105652	1
```

ExcelやGoogleスプレッドシートにそのままペーストして使えます。

## インストール

Chrome Web Store には公開していません。以下の手順でローカルインストールしてください。

1. このリポジトリをクローン（またはZIPでダウンロード）

   ```bash
   git clone https://github.com/chibiegg/chrome-extension-akizuki-copy.git
   ```

2. Chrome で `chrome://extensions/` を開く

3. 右上の「デベロッパーモード」を有効にする

4. 「パッケージ化されていない拡張機能を読み込む」をクリックし、クローンしたフォルダを選択

## 使い方

1. [秋月電子の買い物かご](https://akizukidenshi.com/catalog/cart/cart.aspx) を開く
2. ツールバーの拡張機能アイコンをクリック
3. ポップアップにコピー内容のプレビューが表示される
4. 「クリップボードにコピー」ボタンをクリック

## ファイル構成

```
.
├── manifest.json   # 拡張機能の設定（Manifest V3）
├── popup.html      # ポップアップ UI
└── popup.js        # カートデータの取得・コピー処理
```
